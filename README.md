# FastAPI Backend Project

A backend application built using FastAPI with authentication, database integration, and background task support.

## Problem Statement
The goal of this project is to develop a scalable and maintainable backend system that supports user authentication, database operations, and asynchronous task processing.

## Features
- JWT-based authentication
- PostgreSQL database integration
- Redis caching
- Background task processing with Celery
- Docker and Docker Compose setup

## Installation

1. Clone the repository:
git clone https://github.com/your-username/your-repo.git

2. Navigate to the project directory:
cd project-root

3. Install dependencies:
pip install -r requirements.txt

4. Run the application:
uvicorn main:app --reload

## Usage
Once the server is running, open your browser and go to:
http://127.0.0.1:8000
